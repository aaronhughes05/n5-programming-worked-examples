Example 2

Running Total

1. Problem Description

Calculate the total cost of 5 items entered by a user.

2. Prediction Activity

How many times will the loop repeat?

3. Step-by-Step Implementation

Step 1: Initialize the total.

total = 0

Step 2: Create a loop for 5 items.

for counter in range(5):

Step 3: Add input to total.

  price = float(input())
  total = total + price
total = 0
for counter in range(5):
    price = float(input("Price: "))
    total = total + price
print(total)
