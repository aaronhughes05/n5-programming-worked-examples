1. Problem Description

Input 5 scores, store them in a list, then traverse the list to count how many passes were achieved and identify the highest score.

This example focuses on array traversal and list processing, as required by the National 5 course specification.

2. Learning Objective

By the end of this example, you should be able to:

  • Store multiple values in a list
  • Traverse a list using a loop
  • Apply conditional logic during traversal
  • Produce a clear output summary from processed list data
