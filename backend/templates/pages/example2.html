1. Problem Description

Calculate the total cost of 5 items entered by a user.

Intended Learning Outcomes

Learning Objectives

By the end of this activity, you should be able to:

  • Identify the required input, process, and output for a running-total task.
  • Use a fixed-count loop to control how many values are processed.
  • Update an accumulator correctly on each pass through the loop.
  • Test and explain how program flow affects the final output.

Teacher expects: students can implement and justify a correct loop-and-accumulator flow with accurate traced totals.
